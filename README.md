# IntelliThreat System 🔐🤖

An AI-powered Insider and Vendor Threat Detection System using Unsupervised Machine Learning for Fin Tech SMEs.

---

## 📌 Overview

IntelliThreat System is designed to detect abnormal and risky behavior from employees, vendors, and contractors in Fin Tech organizations. It learns normal user activity patterns and generates risk scores to identify potential insider threats before major damage occurs.

This system is especially useful for small and medium enterprises that do not have dedicated security teams.

---

## 🎯 Key Features

- ✅ Unsupervised anomaly detection
- ✅ Risk score generation (0–1 scale)
- ✅ Real-time alert system
- ✅ Explainable AI insights
- ✅ Interactive admin dashboard
- ✅ Secure user management

## ⚙️ Installation

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/IntelliThreat-System.git
   ```

2. Navigate to the project directory

   ```bash
   cd IntelliThreat-System
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```
